🌱 Agri-Tech – Smart Precision Agriculture System

This project is an intelligent precision agriculture system that combines Artificial Intelligence and graph-based optimization to improve farm management. It models agricultural fields as graphs where each zone is analyzed to detect plant health conditions such as disease or water stress.

The system optimizes robot routing using the Traveling Salesman Problem (TSP) and designs efficient irrigation networks using Minimum Spanning Tree (Kruskal algorithm). In addition, a Graph Neural Network (GNN) is used to predict the spread of diseases and environmental stress, while an NLP module interprets weather reports into actionable farming decisions.
